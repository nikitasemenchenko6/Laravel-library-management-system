# Laravel Library Management System

## Quick Start 
1. clone the repo
2. cd Laravel-libraray-management-system
3. composer install
4. cp or copy .env.example .env
5. php artisan key:generate
6. create database 
7. php artisan migrate
8. php aritsan db:seed
9. php artisan serve
